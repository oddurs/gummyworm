# gummyworm

> Transform images into glorious ASCII art

- 256-color and true color (24-bit) terminal output
- Animated GIF support with terminal playback
- 12+ built-in palettes including emoji and braille
- Export to HTML, SVG, PNG, GIF, or plain text
- Batch processing with recursive directory support
- Works on macOS, Linux, and WSL
